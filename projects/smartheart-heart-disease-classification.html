SmartHeart: Heart Disease Classification

Team machine learning project built to analyze structured clinical data and support heart disease prediction with a workflow that could be explained clearly to both technical and non-technical audiences.

  • 920 Patient Records
  • 4 Models Compared
  • Python Workflow
  • Slides + Report + Code

What Was Done

  • Prepared the UCI heart disease dataset by cleaning fields, treating impossible zero values as missing, encoding categories, and scaling features.
  • Built exploratory analysis around age, cholesterol, maximum heart rate, oldpeak, and other clinically relevant indicators.
  • Trained and compared Logistic Regression, Decision Tree, KNN, and Random Forest models with oversampling and tuning.
  • Documented the methodology and business impact in a formal report and presentation.

Why It Matters

  • Connected technical modeling work to a healthcare decision-making use case.
  • Balanced model comparison, interpretability, and communication.
  • Showed end-to-end project structure from data cleaning through reporting.
